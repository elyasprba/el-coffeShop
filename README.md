## üìå How To Install?

-  Clone This Repo

```
$  git clone https://github.com/elyasprba/el-coffeShop-backend.git
```

-  Go To Folder Repo

```
$  cd el-coffeeshop
```

-  Setup .env

```
DB_USER
DB_HOST
DB_DATABASE
DB_PASSWORD
DB_PORT
DATABASE_URL
JWT_SECRET
JWT_ISSUER
CLOUD_NAME
CLOUD_API
CLOUD_SECRET
```

-  Install Module

```
$  npm install
```

-  To Start The Project

```
$  npm run startDe
```
